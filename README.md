# Calculadora de Emissões

Sistema para cálculo de emissões de gases de efeito estufa conforme padrões internacionais.

## Sobre o Projeto

Esta API fornece endpoints para cálculo preciso das emissões de carbono, seguindo as diretrizes do GHG Protocol e outras regulamentações ambientais.

## Características Principais

- Cálculo dos três escopos de emissão
- Fatores de emissão atualizados
- Relatórios detalhados
- Integração com sistemas existentes

## Requisitos

- Python 3.8 ou superior
- Docker (opcional)
- Conexão com internet para fatores de emissão atualizados
